## ⚡ What is this?

Identity Breach Lab is a cyber attack simulation platform that visualizes how attackers move across enterprise identity systems.

It combines:

- 🔗 Graph intelligence
- ⚔️ Attack chaining
- 🎬 Replay engine
- 📖 Narrative attack storytelling

👉 to simulate real-world identity breaches end-to-end

## 🧠 Why this matters

Modern attacks don’t start with exploits.

They start with:

- stolen credentials
- weak permissions
- over-privileged identities

This project models exactly that:

Identity → Privilege → Access → System → Crown Jewel
