## ⚡ What is this?

A **high-end Threat Intelligence platform** built as a frontend-only product.

It simulates how analysts:

- enrich IoCs
- correlate signals
- map adversary behavior
- produce intelligence reports

---

## 🎯 Why this project?

Most cyber dashboards look like SOC tools.

This one is different:

> 🧠 Built as a **CTI analyst platform**
> 🎨 Designed like a **premium SaaS product**
> 🔗 Structured as a **modular ecosystem component**

---

## 🔥 Key Capabilities

### 🧠 Intelligence Engine

- Multi-source IoC enrichment (mocked feeds)
- Threat scoring & confidence model
- Correlation logic across sources, tags & campaigns
- Indicator profiling (IP, domain, hash, email, actor)

---

### ⚔️ Correlation & Analysis

- IoC comparison engine (side-by-side)
- Shared intelligence detection (tags, campaigns, actors)
- MITRE ATT&CK mapping
- Threat clustering & attribution

---

### 🧭 Analyst Workflow

- Threat hunting mode (hypothesis-driven)
- Investigation timeline
- Relationship graph (IoC → Campaign → Actor)
- Pivot-based navigation

---

### 🗂️ Analyst Memory Layer

- Saved investigations
- Watchlist tracking
- Local notes system
- Persistent workbench

---

### 📄 Intelligence Output

- Exportable PDF reports
- Executive-style summaries
- Analyst recommendations
- Investigation narratives

## 🛰️ Ecosystem Integration

This platform is designed to plug into a larger system:

### Purple Team Lab Integration

```txt
/?incident=incident-001&ioc=secure-login-support.com&ip=185.77.44.21&returnTo=https://purple-team-lab.vercel.app
```

### Behavior

- Auto-load IoC from external module
- Display linked incident context
- Generate correlated intelligence
- Allow return navigation to source platform
